# BookApp

An experimental project exploring better ways to organize and discover literature.

The idea is to build a system where users can track books, organize them with tags, and potentially discover new books based on themes, mood, or style rather than simple genres.

## Current Focus

Right now the project focuses on building a small backend system that retrieves book metadata from external APIs and stores it locally.

This will allow experimentation with:

- tagging systems
- search and filtering
- future recommendation ideas

## Planned Features

- Add books to a TBR (To Be Read) list
- Store metadata such as author, language, and publication year
- Tag books by themes or categories
- Track books that have been read
- Add notes or short reviews

## Possible Future Ideas

- More flexible search across themes and tags
- A system for discovering books based on literary qualities
- AI-assisted recommendations based on style, tone, or themes

## Tech Stack (planned)

- Python
- OpenLibrary API
- SQLite
